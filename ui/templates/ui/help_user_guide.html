Руководство пользователя

Коротко: куда нажимать и что где искать.
Навигация
  • Товары — справочник товаров и работа с карточками.
  • Поставки — учет поступлений и поставок.
  • Выручка — данные продаж и агрегаты.
  • Заявки — заявки и статусы обработки.
  • Склад магазина — остатки по магазинам.
  • Справочники — вспомогательные сущности.
  • Аналитика — сводные метрики/отчеты.
  • SQL — выполнение SELECT-запросов (если разрешено правами).
Права доступа
Пункты меню отображаются только при наличии прав (permissions). Если нужного пункта нет — значит, роль не выдала доступ.

Основные сценарии

1) Просмотр списков
Откройте нужный раздел в меню сверху. В таблице используйте сортировку/поиск (если реализовано на странице).
2) Создание / редактирование
На страницах сущностей используйте кнопки добавления/редактирования. Если кнопок нет — вероятно, нет прав на изменение.
3) SQL-консоль (только SELECT)
В разделе SQL можно вводить запросы и получать результат в виде таблицы. Экспорт выгружает текущий результат в файл (если это включено в твоей реализации).

Горячие ответы

403 — нет прав доступа. Проверь роль/permissions.
404 — страница не найдена. Проверь ссылку/маршрут.
Нет пункта меню — пункт скрыт, пока нет прав или пользователь не суперюзер.
